Syllabus

Project Title: BlogBase Database for Current Affairs
Total Points: 60
Learning Outcomes:
By the end of the course, students are expected to be able to model, design, implement and deploy a BCNF/3NF relational database application on a server for the end-users complete with web-based user interfaces.
Competencies Earned:
Students will learn to use Flask, MySQL, CKEditor and integration of PDF based newsletter authoring and editing tools.
 
Description of the Project:
In this project, you will design a blogging database in which readers and writers could come together to discuss current affairs. An editorial board will moderate the submissions and thus will have the authority to edit or disallow content written by an author, or retract contents. Readers will be able to comment on articles, and on other readersâ€™ comments in ways similar to Facebook. All the articles will have to be presented in the form of a newsletter in PDF with active links to multimedia contents such as pictures, videos, sounds and other types of contents.
